# Práctica 4: Página web estática
## Objetivo de la práctica
Empleando los conocimientos de HTML+CSS vistos en clase (y apoyándose en la documentación de Mozilla), se creará una web sobre un carrito de compra. La web constará de una página principal y 3 páginas interiores interconectadas. Las páginas que componen la web deben contener elementos diversos de HTML:
* Títulos
* Párrafos
* Tablas
* Multimedia
* Formularios Web

La web deberá tener un estilo personal y uniforme utilizando CSS.
