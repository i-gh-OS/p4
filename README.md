# Practica 4: Pagina web estatica

Esta practica consiste en crear una web sencilla sobre un carrito de compra usando HTML y CSS.

La pagina esta formada por:
- una pagina principal
- una pagina de productos
- una pagina de carrito
- una pagina de contacto

La web incluye varios elementos vistos en clase:
- titulos y parrafos
- tablas
- imagenes y contenido multimedia
- formularios web
